웹표준가이드 CSS 규칙

1. CSS란?
  • css란 Cascading Style Sheet이다. HTML문서에 작성한 표준안에 맞는 태그, 그리고 태그명에 입력한 아이디,클래스명을 선택자로, 다양한 값을 적용하여 사용자들에게 다양한 시각적 경험을 제공한다.
2. 선택자
  • css작성의 선택자로는 대표적으로 아이디,클래스,요소 선택자가 있다.
  • 아이디 선택자 예제 확인하기

    아이디는 #을 붙여서 선택 (본 사의 HTML 규칙에 의거, 아이디는 대문자로 시작)

    #Id {margin:0}
    div#Id {margin:0}

  • 클래스 선택자 예제 확인하기

    아이디는 .을 붙여서 선택 (본 사의 HTML 규칙에 의거, 클래스는 소문자로 시작)

    .class {margin:0}
    div.class {margin:0}

    Do you want to close the layer? [Y]es, I agree

  • 요소 선택자 예제 확인하기

    ul {list-style:none}
    ul li {margin:0}

    Do you want to close the layer? [Y]es, I agree

  • 근겁선택자, 가상선택자 예제 확인하기

    div,span,ul,ol,dl => 명시되어 있는 태그들을 다중으로 선택
    div+p => div 바로 다음에 오는 p 요소 선택
    div~p => div 마지막 p까지의 해당되는 태그의 형제요소들 선택
    ul li:first-child => ul li의 첫번째 요소 선택
    ul li:last-child => ul li의 마지막 요소 선택
    ul li:nth-child(n) => ul li의 n의 값을 입력하여 이에 해당하는 요소 선택
    ul li::before => ul li가 생성 되기 전에 생성된 요소 선택
    ul li::after => ul li가 생성 되고 난 후 생성된 요소 선택

3. 작성순서
  • css의 많은 값들의 작성순서는 레이아웃에 크게 영향을 주는값들부터 소소한값으로, 레이아웃에 영향을 주지 않는 요소 순으로 작성한다. 작성순서는 아래의 표를 참조 (모든 값을 명시한 건 아니며, UI개발자들간의 토론을 통해 언제든 개선이 가능)
  • 속성특징 확인하기
    CSS 속성명, 속성특징
    속성명 속성특징
    z-index 3d측면으로 봤을때 요소들이 포개어지는 요소들의 우선순위
    display 요소의 시각적,형태에 대한 값
    overflow, visibility 요소의 영역을 넘치는 것을 자를지의 여부와, 시각적으로 보여지는 여부만을 설정하는 값
    position 요소를 기준점으로 정할지, 기준점으로부터 위치를 시킬지를 설정하는 값
    float, clear 요소를 띄울 지, 띄워진 값에 대한 값에 영향 및 승계에 대한 여부를 설정 하는 값
    width, height 요소의 넓이값, 높이값을 설정하는 값
    padding 요소 내부의 여백을 설정하는 값
    margin 요소 외부의 여백을 설정하는 값
    font font-weight,font-size,line-height,font-family
    등의 값이 있으며, 글자의 굵기,크기,행간,글꼴을 설정하며 font:bold 0/0 dotum; 과 같이 축약형으로 설정이 가능
    letter-spacing line-height가 행간으로써 줄과 줄 사이의 간격을 설정하였다면, 해당 값은 글자와 글자의 사이, 자간을 설정하는 값
    text-align 문장에서의 텍스트의 정렬위치를 설정하는 값
    text-decoration 글자의 꾸밈을 설정하는 값
    text-overflow 글자의 잘림(... 말줄임처리)을 설정하는 값
    white-space, word-wrap, word-break 문장이 길이를 넘게 될 시, 한줄로 보여질 지의 여부를 설정하는 값 문장의 경우 한단어를 하나의 덩어리로 인식하여 의도치 않게 줄바꿈이 될 경우가 있다. 해당 속성으로 해제가 가능하다
    border 요소에 테두리의 모양,굵기 등을 설정하는 값
    color 요소의 글자색상을 설정하는 값
    background 요소의 배경을 설정하는 값

4. 작성유의점
  • 서비스 및 블럭단위로 주석을 표시해 용도를 구분해주고, 선택자 {속성:값;속성:값} 과 같이 작성하며, 각각의 속성의 구분은 ;(세미콜론)으로 구분하고, 선택자 와 값사이의 공백은 허용하되, 속성사이의 공백이 들어가지 않도록 유의한다.
  • 예제 확인하기

    /* 공백없이 붙이며, 맨 마지막 세미콜론은 제거 */

    /* layout */
    #Header,#Container,#Footer {position:relative}

    #Id { position : relative } (X) => #Id {position:relative} (O)
    .class {position : relative;} (X) => #Header {position:relative} (O)

5. CSS 아이콘 및 모션
  • 비교적 간단한 아이콘은 css로 구현하여, 아이콘의 갯수는 점차 늘려갈 예정이며 이미지보다 css로 구현하려는 이유는 다양한곳에서 취향에 맞게끔 컬러값, 굵기 ,크기 등을 수정할 수 있으며 사이트의 request를 줄여 성능향상을 기대코자 하는 의도이다. 또한 스크립트로 애니메이션같은 효과를 줄 시에, 스크립트로 모션제어를 하는게 아닌, css를 사용하여 모션을 제어한다면, 스크립트보다 부드럽고 자연스럽게 동작하는 이유도 있다.
